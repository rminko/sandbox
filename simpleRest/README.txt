Task objective
=====================================
Implement REST Service for users Sign In/Sign Up
using Spring MVC, Hibernate Criteria API, HSQL



First step was: generate maven project using
AppFuse Basic Spring MVC Archetype



To get started, please complete the following steps:

1. Download and install a MySQL 5.x database from 
   http://dev.mysql.com/downloads/mysql/5.0.html#downloads.

2. Run "mvn jetty:run" and view the application at http://localhost:8080.

3. More information can be found at:

   http://appfuse.org/display/APF/AppFuse+QuickStart
